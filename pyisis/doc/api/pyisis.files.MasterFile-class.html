Class MasterFile

source code

object --+
         |
        MasterFile

Encapsulates a traditional CSD/ISIS Master File. Stores the control information for the file.

If load_xrf is set to True (default True), then the MasterFile instance will try to load the whole xrf file into memory.

_load_xrf(self)

source code 
Read self.xrf_fd file and try to load into memory a table of XrfRecord instances. This should be controlled by the load_xrf parameter at instance construction. If load_xrf is set to false, the whole .xrf file will not be loaded into memory, but read on demand.
