executor_loop(context)

source code 
This function expects to receive a context object, having a pipeline and an execution as attributes. Each pipeline element is either a resource or an action. While iterating through the pipeline, each time a resource is found it is pushed into the execution's stack. If an action is found it pops its parameters from the execution stack, optionally pushing its results on to the top of the stack.
